# 🧱 Structs & Enums

## 📐 Structs

```rust
struct Point {
    x: f32,
    y: f32,
}
```

```csharp
struct Point {
    public float X;
    public float Y;
}
```

## 🎯 Enums

```rust
enum Direction {
    Up,
    Down,
    Left,
    Right,
}
```

```csharp
enum Direction {
    Up,
    Down,
    Left,
    Right
}
```

---

## 🧪 Example Usage

```rust
let d = Direction::Left;
```

```csharp
Direction d = Direction.Left;
```
